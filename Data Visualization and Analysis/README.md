# Data Visualization Using Health Dataset

## Objective
To analyze and visualize health-related data using different data visualization techniques.

## Dataset
Health_heart_experimental.csv

## Tools Used
Python

Pandas

Seaborn

Numpy

Matplotlib

Vs Code

Jupyter Notebook


## Visualizations Included
1. Line Chart - Average Blood Pressure vs Age
2. Pie Chart - Counts of Healthy and Clinically risk individuals
3. Bar Plot - Distribution of BMI into different categories 
4. Box Plot - Relationship between height,sex and health status
5. Scatter Plot - Weight Distribution for different heights
6. Histogram - Age Distribution

## Conclusion
Different visualization techniques were used to explore trends, distributions,relationships among various health variables. And, later on they were analyzed to extract meaningful insights into cardiovascular and physical attributes of individuals.
